# Wauble Starter Theme

Wauble Starter Theme is an attempt at a minimal-yet-helpful Wordpress starter theme.

## Installation

```bash
git clone git@github.com:james0r/wauble.git
cd wauble
yarn install
```

## Usage

```bash
yarn dev
```

## Features

- Gulp Build System
- Sass compilation
- ES6 transpilation
- A custom toast that shows what template is used while logged in as admin
- BrowserSync reloading with CSS hot-reloading.
- CMB2 meta field library
- Dynamic sections functionality with a modules manager for admin posts/pages
- Sass 7-1 (ish) file structure.
- A functions.php inspired by Wordpress' TwentyTwentyOne theme.
